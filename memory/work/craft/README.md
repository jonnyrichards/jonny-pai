# Work Craft Knowledge

This directory stores curated knowledge about *how you work* at Culture Amp — not tied to specific projects, but about your craft and communication style.

## Files

- `exec-comms-examples.md` — Real examples of Culture Amp executive communications (Slack-native style)

## Purpose

**Craft knowledge is different from project knowledge:**

| Type | Storage | Purpose |
|------|---------|---------|
| **Craft** | `memory/work/craft/` | How to write, communicate, work (timeless style) |
| **Project** | `memory/work/{project}/` | What you're working on (temporal context) |

**Examples:**
- "How to write exec comms at Culture Amp" → craft
- "Voice project rollout timeline" → project (voice/)
- "How to structure RFCs" → craft
- "Voice beta labeling decision" → project (voice/)

## Related Skills

Craft knowledge often pairs with skills that need examples:
- `/exec-comms` → references `exec-comms-examples.md`
- Future: `/rfc`, `/one-pager`, `/strategy-doc` could all reference craft examples

## Adding Examples

When you write something good, add it here:
1. Create or update the relevant craft file
2. Include the example with context
3. Note what made it work
4. Update the related skill to reference it
