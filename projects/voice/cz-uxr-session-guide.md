# Coach Voice — CZ Research Session Guide

## Opening
1. Before we start — just to set expectations: Voice is currently a desktop-only feature. How does that land for you? Is desktop where you'd naturally use Coach for coaching conversations?

## First impressions (dictation mode)
2. We're going to start with *dictation mode* — voice input, text output. Go ahead and start a coaching conversation on a topic that's real and relevant to you right now. Use it for 3–5 minutes. *(Observe: do they find the controls, any hesitation, any drop-off)*
3. How did that feel? Was the UI clear — did you know how to start, stop, and navigate without being told?
4. How did the response speed feel? Did it ever break your flow or interrupt your thinking?

## Conversation mode
5. Now we'd like you to try *conversation mode* — voice in, voice out. Same idea: pick a topic and just go. *(Observe: reaction to the voice, any latency friction, drop-off points)*
6. What was your immediate reaction to the voice itself — tone, pace, personality? Did it feel right for a coaching context?
7. Compared to dictation, which mode felt more natural? Can you see yourself using both, or defaulting to one?

## Voice preference
8. We're trialling a few different voices. Listen to these three short samples. *(Play samples)* Which feels most appropriate for professional coaching, and why?

## Utility & barriers
9. Are there situations where you'd actively *not* want to use voice — open office, sensitive topics, something else?
10. What would need to be different for voice to become a regular part of how you use Coach?

## Close
11. If you could change one thing before this goes to a wider audience, what would it be?
